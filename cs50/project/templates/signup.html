Thank you for signing up with Chore Log.



Creating a house profile


Please enter a name for your house in the address box. Street addresses are the obvious choice, but a nickname is fine too.

A unique house profile will be created. A registration code will be emailed that will let you create a user account linked to your house.

If you live with others you can share the registration code so that they can also log their chores.

If a house profile has already been set up you do not need to sign up for a new one. Please use the registration code that has already been issued to create your user account at https://chorelog.onrender.com/regcode



Your email address is used to send your registration code but will not be stored until you register a user account
